Client Manager 已成功部署到您的 Kubernetes 集群中！

1. 获取应用的访问方式：

   您可以通过以下命令查看 Service 的详细信息：
   kubectl get svc --namespace {{ .Release.Namespace }} {{ include "client-manager.fullname" . }}

2. 访问应用：

   如果您的集群支持 LoadBalancer 类型的 Service，您可以使用以下命令获取外部 IP：
   kubectl get svc --namespace {{ .Release.Namespace }} {{ include "client-manager.fullname" . }} -o jsonpath='{.status.loadBalancer.ingress[0].ip}'

   如果使用的是 ClusterIP，您可以通过端口转发来访问：
   kubectl port-forward --namespace {{ .Release.Namespace }} svc/{{ include "client-manager.fullname" . }} 8080:{{ .Values.service.port }}

3. 查看应用状态：

   kubectl get pods --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "client-manager.name" . }},app.kubernetes.io/instance={{ .Release.Name }}"

4. 查看应用日志：

   kubectl logs --namespace {{ .Release.Namespace }} -f deployment/{{ include "client-manager.fullname" . }}

5. 健康检查：

   应用提供了 /healthz 端点用于健康检查，您可以通过以下方式访问：
   curl http://localhost:8080/healthz

6. Swagger 文档：

   应用提供了 Swagger API 文档，可以通过以下地址访问：
   http://localhost:8080/swagger/index.html

祝您使用愉快！
